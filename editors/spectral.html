Command Popup Help
Spectral Keystroke Commands
Use '⌘' instead of 'Ctrl+Alt' on Mac
  Ctrl+c         Copy
  Ctrl+x         Cut
  Ctrl+v         Paste
  Ctrl+s         Save file
  Ctrl+i         Insert HTML (e.g. for video embedding)
  Ctrl+m         Insert note
  Ctrl+l         Insert audio recording
  Ctrl+q or Esc  Javascript command evaluator popup
  Ctrl+b         Toggle toolbar visibility
  Ctrl+h         Show this help
  Ctrl+[1-7]     Send selected text to the n-th search box
  Ctrl+Alt+1     Add explanation around selection
  Ctrl+Alt+l     Record webcam video into the editor
  Ctrl+Alt+m     Match brackets (+ braces and parenthesis)
  Ctrl+Alt+p     Scale selected or nearby picture(s)
  Ctrl+Alt+P     Iconize selected picture(s)
  F7             Diff two notes
  F8             Pick and copy token under click
  F9             Annotate nearest image
  Ctrl+Alt+/     Search for multiple regexes (one per line)
  Ctrl+Alt+S     Save state
  Ctrl+Alt+z     Undo to the last saved state
  Ctrl+Alt+y     Redo undone state reversal 
  Ctrl+Alt+c     Clear temporary highlighting
  Ctrl+Alt+v/V   Paste accross lines at starts(v)/ends(V)
  Ctrl+Alt+d     Delete lines with cursors
  Ctrl+Alt+k     Keep only lines with cursors
  Ctrl+Alt+D     Delete lines with highlighting
  Ctrl+Alt+K     Keep only lines with highlighting
  Ctrl+Alt+q     Add attachment
  Ctrl+Alt+s     Save editor content locally
  Tab            Indent selection
  Shift+Tab      Unindent selection
Text Commands (Can be used with Javascript Eval):
  cmdhelp      → Show command popup command reference
  wc           → Word count of selection
  reflow(n)    → Reflow selected text to n-character lines
  savesvg      → Save editor SVG content to an SVG file
  svgrotate(n) → Rotate inline SVGs counterclockwise by n degrees
  svgscale(n)  → Scale inline SVGs by factor n around their centres
  startVideoRecording() → Record webcam video into the editor
  delgraphics  → Delete SVG elements overlapping 2-point rectangle
  delgraphics_included → Delete SVG elements fully inside 2-point rectangle
  delete_empty_lines → Delete blank/whitespace-only lines
  camel        → Convert identifier to camelCase
  block_color(color, regexes) → Color matching top-level brace blocks
  nested_block_color(level, color, regexes) → Color matching nested brace blocks
  diffnotes()  → Select two notes and show a diff
  setDiffContext(2,2) → Show 2 context lines around note diffs
  clearDiffContext() → Show full note diffs
  searchnotes("text") → Search embedded notes
  snake        → Convert identifier to snake_case
  kebab        → Convert identifier to kebab-case
  uc           → Convert to upper case
  lc           → Convert to lowercase
  title        → Convert to Title Case
  darken       → Switch to dark mode
