HLT 3-way demo inputs
=====================

Files:
- ancestor.hlt
- yours.hlt
- theirs.hlt

Launchers at repo root:
- launch_hltdiff_3way.sh
- launch_hltdiff_3way.bat

Merge output path preconfigured by launchers:
- hltdiff_demo_3way/merged_output.hlt

How to save merge output in hltdiff:
1. Make per-diff merge choices with toolbar buttons:
   - L, R, LR, RL
2. Save via one of:
   - Menu: Merge -> Write Merge File...
   - Merge preview window: Save (or Save & Exit)

If launched with -o, the save target is already set to merged_output.hlt.
